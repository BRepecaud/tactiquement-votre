FONDAMENTAUX

  • Troisième homme
  • Jeu sans ballon
  • Prise d'information

Le troisième homme, c'est le dernier angle du triangle, le résultat d'un mouvement de passe qui a permis à ce joueur d'être libre de tout marquage.
La recherche de ce joueur est essentielle, c'est le liant de chaque action, le joueur qui crée le surnombre, le décalage.

Le porteur de balle est essentiel dans la création d'une action, mais les mouvements autour de lui le sont autant voire plus.
Le jeu sans ballon crée des espaces, des zones que le porteur de balle peut utiliser.

La marque des plus grands est leur capacité à voir le jeu avant tout le monde. Mais au-delà du talent, pour avoir ce temps d'avance sur les autres, la prise d'information est primordiale. "Scanner" le jeu, les joueurs qui les entourent, avant de recevoir le ballon. Cela leur permet de contrôler le temps, d'être à même d'accélérer le jeu d'un seul coup d'oeil.
